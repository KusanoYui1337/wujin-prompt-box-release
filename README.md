# 五金AI提示词盒子

五金AI提示词盒子，是一个用于保存、分类和管理提示词的小工具。

适合经常使用 AI 写作、编程、绘图、办公的用户，把常用提示词统一保存起来，避免每次都重新翻聊天记录、复制旧内容。

## 软件用途

- 保存常用提示词
- 按分类管理不同用途的提示词
- 快速复制提示词内容
- 方便整理 AI 写作、AI 编程、AI 绘图等常用指令
- 适合个人日常使用
- 支持多国语言
